Виды квантизации

Как расположить уровни?

Квантизация = дискретизация

Непрерывный сигнал → конечное число уровней

Две оси классификации

  • Скалярная vs Векторная
  • Uniform vs Non-uniform

Равномерная (uniform)

$$Q(x) = \text{clamp}\!\left(\text{round}\!\left(\frac{x}{s}\right) + z,\; 0,\; 2^b{-}1\right)$$

$$\hat{x} = s \cdot (Q(x) - z)$$

Параметры

  • Scale $s = \frac{x_{\max} - x_{\min}}{2^b - 1}$ — шаг сетки
  • Zero-point $z$ — смещение

Это INT8 / INT4 в индустрии

Неравномерная (non-uniform)

Сетка сгущается в областях, где значений больше — обычно у нуля.

Float-форматы — логарифмическая сетка: мелкие числа представлены точнее.

Примеры

  • OFP8 E4M3 — на sim8 в хакатоне
  • FP8 E5M2 — шире dynamic range
  • BF16 — Google Brain
  • NF4 — оптимальная для $\mathcal{N}(0,1)$

Кластеризация (k-means)

Не навязывать сетку, а найти оптимальные уровни из данных.

Хранить: таблица центров ($32 \text{ бит} \times 2^b$) + индексы ($b \text{ бит} \times n$)

Предельные случаи

  • Binary $\{-1, +1\}$ — 1 бит
  • Ternary $\{-1, 0, +1\}$ — 1.58 бит
  • LUT — любые $2^b$ значений в таблице

Проблемы квантизации

Outlier'ы

Outlier'ы растягивают диапазон

$[0.1,\; 0.3,\; 0.2,\; 0.4,\; 0.15,\; 0.25,\; 0.35,\; 47.0]$

range = 46.9 → scale ≈ 0.184
Числа 0.1–0.4 → уровни 0–2

Результат

Один outlier убил точность для 7 из 8 чисел.

Ошибка 50–100% для нормальных значений

Квантизация в нейронных сетях

Чем особенная?

Избыточность

Миллионы параметров кодируют простые закономерности. Шум квантизации — ещё один вид регуляризации.

Гладкость loss landscape

Малые изменения весов → малые изменения выхода. Нейронная сеть — не система уравнений, а статистическая модель.

PTQ

Post-Training Quantization

  • После обучения
  • Нужна модель + калибровочные данные
  • Минуты
  • Хорошо до 8 бит

QAT

Quantization-Aware Training

  • Во время обучения
  • Нужен тренировочный пайплайн
  • Часы–дни
  • Лучше при 4, 2, 1 бит
